# SaultJs
---
`salutjs`是一种为快速搭建H5单页应用的脚手架。它的底层代码是建立在backboneJS上的。通过对backbone进步一封装和改进，将创建单页应用的过程简单化。通过一些简单的配置快速建立单一界面应用。一下是它的一些使用方法。
#####下载项目执行安装依赖包
```javscript
npm install
```
在`node`目录下打开执行以下命令
#####启用数据服务
```javascript
node json
```
#####运行服务程序
```javascript
node run
```
####在浏览器中打开
`loaclhost:8080/page_html#A`
